# Analisis-de-Rendimiento-Hist-rico-y-Moderno-en-F-rmula-1
Desarrollé un reporte interactivo estructurado en dos ejes: el impacto histórico de pilotos nacionales y la evolución competitiva en la era moderna (2014-2024).
Implementé procesos de modelado y transformación de datos, destacando la estandarización de métricas históricas al reglamento actual mediante expresiones DAX.
Impacto: Facilitó la comparación objetiva de tendencias a largo plazo, el dominio por escuderías y el seguimiento de KPIs de los principales pilotos.
